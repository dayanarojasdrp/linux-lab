# linux-lab
This repository contains a complete, practical, and progressive lab designed to demonstrate Linux proficiency at the Junior+ level, from the basics to real-world server administration.

It includes:
* User and permissions management
* Advanced shell commands
* Processes and systemd
* Real-world networking
* Compression and bundling
* Advanced cron commands
* Logs and log rotation
* Basic security
* Final integrative project
Each module contains:
* Objectives
* Commands
* Exercises
* Success criteria
* Real-world practice
